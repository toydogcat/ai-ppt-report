# Luna AI PPT Presentation Hub

宇宙級靜態簡報展示站。基於 Web 端的華麗、響應式且具備極致 3D 轉場效果的 AI 簡報展示系統。

## 快速連結
- **線上演示**: [https://toydogcat.github.io/ai-ppt-report/](https://toydogcat.github.io/ai-ppt-report/)
- **上架教學**: 請開啟演示站並尋找「Luna AI PPT 上架與開發教學」簡報。

## 專案特性
- **Markdown 驅動**: 撰寫簡單的 Markdown 即可生成高品質 3D 簡報。
- **極致視覺**: 支援 3D 翻轉、旋轉、縮放等多種轉場特效。
- **功能完備**: 內建雷射筆、自動播放、快捷鍵支援與 KaTeX 數學渲染。
- **純靜態架構**: 零成本部署，極速讀取。
